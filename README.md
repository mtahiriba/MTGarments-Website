# MTGarments-website
E-commerce website that specializes in selling clothing for men, women, and children.

## Technology used
1. HTML 5
2. CSS 3
3. Bootstrap 5

## How to run on local Machine.
1. Make clone of repository on local machine.
2. Open project.
3. Double click on index.html.
